SFML_Button_Library
C++ SFML Button Library  
 bt::Button - SFML Button Library

A modern, feature-rich button library for SFML with grid layouts, animations, icons, tooltips, and more!

 Features

-  Multiple button shapes (Rectangle, Circle, Triangle, Rounded)
-  Hover, Press, and Disabled states
-  Grid Layout System (Rows & Columns)
-  Keyboard shortcuts
-  Tooltips
-  Icons
-  Animations (Bounce, Pulse, Fade, etc.)
-  Callback system
-  Button Groups & Containers
-  Toggle buttons
-  Customizable colors, shadows, outlines
